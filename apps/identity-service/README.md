# Identity Service

Identity and Authentication microservice for managing users and JWT-based authentication.

## Setup

1. Copy `.env` and update database credentials if needed
2. Install dependencies: `pnpm install`
3. Start dev server: `pnpm start:dev`
4. Build for production: `pnpm build`

## Database

PostgreSQL database: `identity-service`

## Swagger Docs

Available at `http://localhost:3004/docs` when running locally.
